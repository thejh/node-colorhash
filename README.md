Make blue midnight color hashes from the full visible spectrum with this cool module! Installable from npm as `colorhash`. Usage:

    colorhash()

Usage:

    > colorhash = require('colorhash')
    [Function]
    > colorhash('TheJH')
    [ 156, 230, 188 ]
    > colorhash('TheJH', 'css')
    '#9ce6bc'
    >
